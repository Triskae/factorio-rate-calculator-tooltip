# Factorio Rate Calculator

A small Factorio mod that displays recipe production and consumption rates using a shared time unit.

Factorio recipes all have their own crafting time. Some recipes also produce multiple outputs, and crafting machines have different crafting speeds. This is accurate, but it makes quick production-chain comparisons harder than they need to be.

The goal of this mod is to make those numbers easier to read, closer to the way Satisfactory presents production: inputs and outputs shown with the same time unit, for example items per minute.

Current features:

- input and output rates in recipe tooltips for assembling machines 1, 2, and 3;
- a small live panel when hovering a crafting machine;
- calculations based on recipe time, machine crafting speed, output quantities, probabilities, and productivity.

Later, the mod should add an in-game setting to choose the display unit, such as per second, per minute, or per hour.
